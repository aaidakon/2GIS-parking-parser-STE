# 🅿️ Парсер парковок 2GIS — Алматы

## 📌 Описание проекта

Проект собирает данные о парковках Алматы через официальный API 2GIS и автоматически выгружает результат в Google Sheets.

В отличие от Selenium или Playwright, используется официальный API 2GIS, который возвращает структурированные данные в формате JSON и позволяет получать информацию значительно быстрее и стабильнее.

---

## 🎯 Цель проекта

Создать актуальный датасет парковок Алматы для дальнейшего анализа, визуализации и использования в BI-системах.

## 🔍 Почему API, а не веб-парсинг?

2GIS построен как SPA-приложение (Single Page Application), поэтому данные загружаются асинхронно через внутренний API.

Использование API позволяет:

- получать данные напрямую в формате JSON;
- избежать проблем с Selenium и headless-браузерами;
- работать быстрее в 10+ раз;
- получать больше атрибутов одним запросом;
- использовать официальный и документированный источник данных.

---

## 🗺️ Стратегия сбора данных

Демо-ключ 2GIS имеет ограничение:

- максимум 5 страниц на запрос;
- максимум 10 объектов на страницу;
- не более 50 объектов на одну точку поиска.

Для обхода ограничения Алматы была разделена на равномерную сетку из 82 точек.

Для каждой точки выполняется поиск парковок в радиусе 3000 метров.

После завершения сбора выполняется дедупликация по уникальному идентификатору объекта (`id_2gis`).

## 🚀 Настройка проекта

### Получение API-ключа 2GIS

1. Перейти на https://platform.2gis.ru
2. Зарегистрироваться
3. Создать Demo Key
4. Вставить ключ в ноутбук

### Настройка Google Sheets API

1. Перейти в Google Cloud Console
2. Создать проект
3. Включить:
   - Google Sheets API
   - Google Drive API
4. Создать Service Account
5. Скачать JSON-файл с учетными данными
6. Переименовать файл в:

```text
service_account.json
```

7. Предоставить сервисному аккаунту доступ к Google Sheets

## ⚠️ Ограничения
